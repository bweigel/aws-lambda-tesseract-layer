Contains ready-to-use binaries for Tesseract for [Lambda runtimes](https://docs.aws.amazon.com/lambda/latest/dg/lambda-runtimes.html) using Amazon Linux 1 and Amazon Linux 2.

## Supported Runtimes

### Amazon Linux 1

- Python 2.7, 3.6, 3.7
- Ruby 2.5
- Java 8 (OpenJDK)
- Go 1.x
- .NET Core 2.1

### Amazon Linux 2

- Python 3.8
- Node.js 10, 12
- Ruby 2.7
- Java 8 (Coretto), 11 (Coretto)
- .NET Core 3.1
